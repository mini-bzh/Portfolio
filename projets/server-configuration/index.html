Configuration d'un serveur

Voici une page de connexion pour administrateurs sur un site web. Comment fonctionne-t-elle et comment peut-on en créer une ? C'est ce que nous avons appris au cours de ce projet.

Description

Ce projet, réalisé durant la première année de BUT Informatique, était un projet d'équipe. Il portait sur la configuration d'un serveur web sur une machine virtuelle totalement isolée d'Internet. Par conséquent, seules des captures d'écran et le compte rendu ont été utilisés pour documenter le projet.

Étapes du projet :

  1. Configuration d'un serveur Apache : Dans un premier temps, nous avons configuré un serveur Apache pour pouvoir afficher un site web statique.
  2. Configuration d'un serveur PHP : Ensuite, nous avons configuré un serveur PHP pour rendre le site web interactif.
  3. Configuration d'un serveur MySQL et PHP : En troisième étape, nous avons configuré MySQL et PHP pour pouvoir interroger une base de données et utiliser des fonctionnalités dynamiques.

Dans notre équipe, nous nous sommes organisés de la manière suivante :

  • Une personne était responsable de la rédaction du rapport.
  • Une personne effectuait les recherches pour comprendre le fonctionnement des serveurs.
  • Quant à moi, j'étais chargé des manipulations sur le serveur lui-même.

Ce projet m'a apporté une compréhension approfondie du fonctionnement d'un serveur web et de son utilisation pratique.
