Steganographie

L'image qui apparait ci-dessus, outre le fait quelle soit très belle, semble n'avoir rien de spécial. Cependant c'est loin d'être le cas. Cette image contient un message dissimulé invisible à l'oeil nu et visible uniquement avec un petit programme que je présente ici.

Description

Ce projet est un projet de première (donc réalisé en NSI au lycée) en autonomie.

Le but était de faire deux fonctions :

- Une première (steganographie()) qui à partir d'un tableau de pixels RVB renvoie ce même tableau avec les pixels très légèrement modifiés de façon à cacher un message dans ces toutes petites modifications.

- Une deuxième (dechiffreur()) qui elle sert à faire le chemin inverse, c'est-à-dire trouver un message caché dans un tableau de pixels.

- J'ai fait une fonction en plus (bruit()) qui permet de modifier très légèrement et aléatoirement les valeurs RVB d'un tableau. Je m'en sert pour éviter qu'une comparaison pixel par pixel entre l'image d'origine et l'image modifiée ne révèle la position des modifications qui cachent le message.
